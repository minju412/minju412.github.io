## CREATE
```
create table member
    (
        id   bigint generated by default as identity,
        name varchar(255),
        primary key (id)
);
```
코드 실행은 `cmd`+`enter`

## INSERT
```
insert into member(name) values('spring')
```

## SELECT
```
select * from member
```
