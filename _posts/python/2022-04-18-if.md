# 조건문
## if문
```py
x=7
if x==7:
    print("Lucky")

x=15
if x>=10:
    if x%2==1:
        print("10 이상의 홀수")
```

## 다중 if문
```py
x=93
if x>=90:
    print('A')
elif x>=80:
    print('B')
else:
    print('F')
```
