## 풀이
### 내 풀이
```py
import sys

W, N = map(int, input().split())
lst = []
for _ in range(N):
    w, p = map(int, input().split())
    lst.append((w, p))

lst.sort(key=lambda x:x[1], reverse=True)

price = 0
i = 0
while W != 0:
    if W >= lst[i][0]:
        W -= lst[i][0]
        price += lst[i][1] * lst[i][0]
    else:
        price += lst[i][1] * W
        W = 0
    i += 1

print(price)
```
