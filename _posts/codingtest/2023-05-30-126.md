## 풀이
### Bottom-Up
```py
N = int(input())
lst = list(map(int, input().split()))
dp = [0] * N # ex. dp[3] 이라면 맨 마지막 원소가 lst[3]인 증가수열들 중 길이가 최대인 증가수열의 길이
dp[0] = 1

for i in range(1, N):
    mx = 0 # ✅ mx는 0으로 초기화 !!
    for j in range(i): # ✅ 내 앞의 원소들을 탐색
        if lst[j] < lst[i]: # ✅ 나보다 값이 작아야 한다 (증가수열을 만들어야 하기 때문)
            mx = max(mx, dp[j])
    dp[i] = mx + 1 # ✅ 나(lst[i])까지 뒤에 붙는거니까 +1

print(max(dp))
```
