# Eqaulity
## == loose equality
```js
const str = '5';
const num = 5;
console.log(str == num); // true
console.log(str != num); // false
```
## === strict equality
```js
const str = '5';
const num = 5;
console.log(str === num); // false
console.log(str !== num); // true
```
