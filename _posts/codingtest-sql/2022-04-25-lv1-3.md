# 이름이 없는 동물의 아이디
## 문제 정리
동물 보호소에 들어온 동물 중, 이름이 없는 채로 들어온 동물의 ID를 조회하는 SQL 문을 작성해주세요. 단, ID는 오름차순 정렬되어야 합니다.
## 내 답
```py
SELECT ANIMAL_ID from ANIMAL_INS where NAME is NULL
```
# 정리
- NULL값인 것, NULL값이 아닌 것 검색
> where 컬럼 is NULL<br>
where 컬럼 is not NULL
