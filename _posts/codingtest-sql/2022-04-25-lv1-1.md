# 모든 레코드 조회하기
## 문제 정리
동물 보호소에 들어온 모든 동물의 정보를 `ANIMAL_ID`순으로 조회하는 SQL문을 작성해주세요.
## 내 답
```py
SELECT * from ANIMAL_INS order by ANIMAL_ID ASC;
```
# 정리
- 데이터 오름차순, 내림차순 조회
> SELECT 컬럼 FROM 테이블 ORDER BY 컬럼 ASC;<br>
SELECT 컬럼 FROM 테이블 ORDER BY 컬럼 DESC;<br>
