# 상위 n개 레코드
## 문제 정리
동물 보호소에 가장 먼저 들어온 동물의 이름을 조회하는 SQL 문을 작성해주세요.
## 내 답
```sql
SELECT name from animal_ins order by datetime limit 1
```
# 정리
- 조회할 레코드의 개수 지정하기
> `limit 10`: 상위 10개 레코드<br>
`limit 5, 10`: 6행부터 15행까지 10개의 레코드
